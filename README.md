# Dashboard de Vendas

Este projeto consiste em um dashboard interativo desenvolvido em Python
para análise de vendas por período e produto.

## Funcionalidades
- Filtros por ano, mês e produto
- Cálculo de faturamento, quantidade vendida e ticket médio
- Gráficos interativos (barra e pizza)
- Interface desenvolvida com Streamlit

## Tecnologias utilizadas
- Python
- Pandas
- Streamlit
- Plotly

## Como executar
```bash
pip install -r requirements.txt
streamlit run app.py
